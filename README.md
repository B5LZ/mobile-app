# mobile-app
This repository contains our code for our mobile version of the Multi-Language Wellness App

# Starting the code 
npm install
npx expo start

# Viewing the app on mobile
1) Download Expo Go on your mobile device
2) If you are on IOS, scan the QR that is produced in the terminal after running npx expo start, and it should open in your Expo Go app.
3) Wait for the app to load and you will see our work.
